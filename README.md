# AI Studio

AI Studio is a fully autonomous AI-powered video production platform.

## Vision

Create a complete production studio using local AI models capable of:

- Researching topics
- Planning productions
- Writing scripts
- Generating narration
- Creating cinematic images
- Generating video
- Performing quality assurance
- Publishing content
